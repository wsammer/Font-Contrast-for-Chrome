How to use Font Contrast

The app has two buttons:

  • The button in the add-on bar toggles the add-on for the current tab. It turns black to indicate activation.
  • The button in the address bar manages the options for the current page.

If the add-on is activated, this text should appear black!

Options guide

Clicking on the options button will open this window:

  • Strength: Adjusts the detection of dark backgrounds. This value can be adjusted to catch more low-contrast text.
  • Increase font (%) / Threshold (px): Enlarges fonts that are below the threshold (in pixels).

  • Whitelist: Adds the website to the whitelist.
  • Blacklist: Adds the website to the blacklist.
  • Advanced mode: Dims text to a darker shade (rather than just black). The effect can be adjusted with the Brightness slider, which appears when this option is activated.

  • Bold text: Changes the font weight of all text to bold.
  • Avoid colorful text: Attempts to avoid colorful text and links, if they are considered readable.
  • Underline black links: Adds an underline to links that turn black, to distinguish them from other text.

  • Force max opacity: Forces full opacity on all fonts.
  • Force black placeholder: Forces all placeholder text to turn black, rather than just increasing their opacity.
  • Input border: Attempts to add a black outline around text areas.

  • Skip headings: Avoids dimming all big headings.
  • Skip white text: Avoids dimming all white text.

  • More options: Opens a tab with all the options, including the ability to activate the add-on everywhere, the global options (with in-depth tooltips), the whitelist/blacklist, and a link to this page.
